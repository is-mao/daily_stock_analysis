# 🚀 股票精选功能快速上手指南

## 🎯 一分钟快速开始

### GitHub Actions 使用（推荐）

1. **进入你的 GitHub 仓库**
2. **点击 `Actions` 标签页**  
3. **选择 `每日股票分析` 工作流程**
4. **点击 `Run workflow` 按钮**
5. **选择运行模式为 `selection-only`**
6. **点击绿色的 `Run workflow` 按钮**

✅ **预期运行时间**: 30-60分钟（大幅优化！）

### 本地使用

```bash
# 基础用法
python main.py --stock-selection

# 自定义数量
python main.py --stock-selection --selection-count 30

# 指定策略
python main.py --stock-selection --selection-strategy trend_following
```

## 📊 2.0版本优化亮点

### 🚀 性能大幅提升
- **运行时间**: 从4小时缩短至30-60分钟 ⚡
- **股票池**: 从4000+只缩减至300-600只 🎯
- **精选质量**: 聚焦热点板块，更有针对性 📈

### 🎛️ 智能筛选策略
1. **获取热点板块**: 实时获取概念板块涨跌幅排行
2. **选择TOP20**: 选择当日表现最好的20个概念板块  
3. **板块内精选**: 每个板块选择前30只股票
4. **市值筛选**: 优先选择50亿-5000亿市值股票
5. **去重优化**: 最终形成300-600只精选股票池

## 🎯 精选策略说明

| 策略 | 适用场景 | 特点 |
|------|----------|------|
| `comprehensive` | 通用（默认） | 多维度综合评分 |
| `trend_following` | 趋势市场 | 多头排列 + 突破 |
| `value_hunting` | 震荡市场 | 低估值 + 基本面好 |
| `momentum` | 强势市场 | 强势股 + 量价配合 |
| `reversal` | 调整市场 | 超跌反弹 + 技术修复 |

## 🔧 故障排除

### 常见问题

**Q: 运行时间还是很长？**
A: 2.0版本已大幅优化，如仍觉得慢可以：
- 减少精选数量：`--selection-count 10`
- 使用趋势策略：`--selection-strategy trend_following`

**Q: 精选结果为空？**  
A: 可能筛选条件过严，系统会自动使用备选股票池

**Q: 如何查看详细日志？**
A: 添加 `--debug` 参数查看详细运行过程
