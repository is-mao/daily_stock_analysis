# 📋 股票过滤规则更新日志

## 🎯 2026-01-21 - 新增920开头股票过滤

### 📝 更新内容

**新增过滤规则**: 在股票精选的二次筛选中，新增对920开头股票的过滤。

### 🔍 技术详情

#### 过滤规则变更
**之前**: 只过滤创业板(300/301)和科创板(688)股票
```python
if code.startswith('300') or code.startswith('301') or code.startswith('688'):
    # 过滤掉
```

**现在**: 新增存托凭证(920)股票过滤
```python
if code.startswith('300') or code.startswith('301') or code.startswith('688') or code.startswith('920'):
    # 过滤掉
```

#### 影响的文件
- `stock_selector.py` - 核心过滤逻辑
- `STOCK_SELECTION_GUIDE.md` - 用户指南
- `CHANGELOG_STOCK_SELECTION.md` - 功能变更日志

### 📊 股票分类说明

| 代码前缀 | 股票类型 | 过滤状态 | 说明 |
|----------|----------|----------|------|
| 600/601/603 | 沪市主板 | ✅ 保留 | 可正常交易 |
| 000/002 | 深市主板/中小板 | ✅ 保留 | 可正常交易 |
| 300/301 | 创业板 | ❌ 过滤 | 需要开通创业板权限 |
| 688 | 科创板 | ❌ 过滤 | 需要开通科创板权限 |
| **920** | **存托凭证** | **❌ 过滤** | **新增：需要特殊权限** |

### 🎯 过滤原因

**920开头股票（存托凭证）特点**:
1. **交易门槛高**: 需要开通存托凭证交易权限
2. **风险较大**: 涉及跨境投资，汇率风险等
3. **流动性差**: 相对于A股主板，交易活跃度较低
4. **适合性要求**: 主要面向专业投资者

### 📈 影响范围

#### 股票精选流程
```
全市场股票池
    ↓ 热点板块筛选
热点股票(~400只)
    ↓ 多维度评分筛选  
精选股票(~20只)
    ↓ 二次筛选(排除300/301/688/920)
可操作股票(前20只) ← 新增920过滤
```

#### 报告生成
- **完整精选列表**: 包含所有符合条件的股票（包括920）
- **可操作股票推荐**: 排除300/301/688/920，适合普通投资者


### 🚀 使用建议

**专业投资者**: 可以参考完整精选列表，根据自己的权限和风险承受能力选择合适的股票。
